Detailed Description

Protocol Options Option Name Description tcp Uses TCP/IP Sockets to connect to the broker. ssl Uses OpenSSL to secure TCP/IP sockets. (Since v3.2.0) failover Composite of other Transports, when a Transport connection fails, the Failover Transport connects to the next Transport in the list or retries the connection to the failed broker URI.
